Hexo异常:Template render error:(unknown path)unexpected end of file

发表于 2021-09-17 分类于 Error ， 环境搭建 ， Hexo 阅读次数：
本文字数： 619 阅读时长 ≈ 1 分钟

¶问题描述

在渲染的过程中遇见了以上问题，我直接就是一顿崩溃。对于某一个页面渲染失败，主要是文章中出现了nunjucks语法的字符。

¶问题定位策略

关于新出现的问题以及定位问题的策略相关

无论是谁在工作、还是日常生活都会出现问题，遇见问题先不要慌，先发个朋友圈？

遇见问题了就去查问题啊，不要怕找不到一模一样的问题，先找一堆相关的问题，然后慢慢来呗。

  1. 根据查到的问题及其解决方案确定出现这类问题的原因
  2. 根据原因回忆你在操作过程中可能出现的类似的情况
  3. 然后对操作进行复盘大致确定问题可能出现的范围
  4. 对范围内的类似情况进行逐一排查

¶问题解决

在网上一查，无论是哪个网站提供的说法不是Hexo无法渲染大括号，还是存在{{}}，{#，'{，{% %}之类的，首先一点是Hexo是支持{%note%}` 标签的，但是我仔细收寻了我的文档，确定以及肯定我的文档当中没有除最后一个之外的其他三个。那就着重找最后一个呗。 对全局的 `{%note%}{%endnote%} 进行排查，这有在这里才会出现Hexo无法转义的字符，或者说成是标签。可能出现的情况就是对于单一标签缺少一个大括号或者一个%，这样的话，使用文本编辑器里面的查找功能，很快就能定位到问题，果然就是缺少一个有括号，问题解决。😉
