3.3.1 字符集命令

Variable_name Description
character_set_server 服务器安装时指定的默认编码格式
character_set_client 主要来设置连接数据库时的字符集。主要来决定MySQL如何解释客户端发送到服务器的SQL指令
character_set_connection 主要来设置连接数据库时的字符集。主要用来决定MySQL如何处理客户端法发来的SQL命令
character_set_result 数据库返回客户端时的使用的编码格式
character_set_system 数据库系统使用的编码格式。pay attention:不要轻易改变它
